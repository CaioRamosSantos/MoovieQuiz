# 🎬 MoovieQuiz

Jogo multiplayer em tempo real onde dois jogadores competem para adivinhar filmes a partir de seus posters. O primeiro a acertar o nome do filme marca o ponto — vence quem tiver mais pontos ao final das rodadas.

**[Jogar agora →](https://moovie-quiz.vercel.app)**

---

## Tecnologias

**Front-end**
- HTML, CSS e JavaScript puro
- Socket.IO Client para comunicação em tempo real

**Back-end**
- Node.js + Express
- Socket.IO para WebSockets
- TypeScript
- API do TMDB para buscar os posters dos filmes

**Deploy**
- Front-end: Vercel
- Back-end: Railway

---

## Como funciona

1. O jogador entra com um nick e procura partida
2. Quando dois jogadores se conectam, a partida começa automaticamente
3. Um poster de filme é exibido — o primeiro a digitar o nome correto marca ponto
4. A partida tem 4 rodadas, o jogador com mais pontos vence
